What Is HTML?
HTML (Hypertext Markup Language) is the standard language used to create and structure content on the web. It allows you to define elements like headings, paragraphs, images and links to structure a webpage
HTML Tags :
1. <html> ... </html> :
The <html> tag marks the beginning of an HTML document and contains all other HTML elements. It is the outermost tag that tells the browser the content is an HTML page
2. <body> ... </body> :
The main section that contains the visible content of the web page. Text, images, links, and interactive elements are placed within this tag. Everything visible to the user is enclosed within the <body> tag
3. <head> ... </head> :
Contains meta information such as the page title, character set, stylesheets (CSS), meta tags (page description, keywords, etc.), and JavaScript. It is not directly visible to the user but is essential for browsers and search engines
4. <title> ... </title> :
Specifies the title of the page, which appears in the browser tab
5. <h1> - <h6> ... </h1> - </h6> :
These are heading tags. <h1> is the most important heading, often used as the main title of the page. <h2>, <h3>, etc., are used as subheadings, creating a hierarchical structure for the content of the page

7. <a> ... </a> :
Creates a hyperlink. This tag allows navigation between pages or to external websites. The target URL is specified with the href attribute
E.G.

8. <img> ... </img> :

Creates an ordered list. List items are defined with the <li> tag, and each item is numbered

12. <div> ... </div> :
A generic container used to group content. It is not visible and takes up the full width, pushing elements below it (it can be styled with CSS)
13. <span> ... </span> :
Used for smaller sections within a document. It is typically used for inline styling..
